What is Redux?
Redux is a state management tool. 

Benefits 
- Predictable state changes
- Centralized state
- Easy debugging
preserve page state undo/redo
- Ecosystem of add-ons
- Is immutable
- State can be accessed anywhere in your applications
- Fast and reliable
- Popular

Disadvantages
- Complexity 
- Verbosity

What problem are you solving?
- When not to use redux:
    - Tight budget
    - Small to medium-size apps
    - Simple UI/data flow
    - static data
